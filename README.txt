
program creates and opens a page containing an album of movies

to run the program:
1. put all 3 python files in the same folder
2. run python file (entertainment_center.py)


program uses 3 files:
1. entertainment_center.py - this file contains movie information
2. fresh_tomatoes.py - this file contains code that generates an html file
3. media.py - this file contains a python class for Movies, to easily store movie information
